# Jupyter Notebook Tips

## ShortCuts

### Jupyter Notebook Navegation Tips

A: adiciona emcima
B: Adiciona em baixo
ENTER: ENtra para editar a célula
DD: Deleta
M: Markdown
Y: Code
ESC: Sai do modo de editar para o de navegaçâo (se estiver no modo de editar)

#### Principais
+ Exec Cell: SHIFT + ENTER
+ Exec and Insert Below: ALT + ENTER
+ Insert Cell Below: ESC + B | B + B (in cell)
+ Insert Cell Above: ESC + A | A + A (in cell)
